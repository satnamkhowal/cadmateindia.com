About Course

Microsoft Project 2016 is a powerful project management software. It is developed and sold by Microsoft, and used in developing a schedule, tracking progress, budget management, and analyzing workloads. Microsoft Project 2016 provides the best set up in linking the tasks together and assign resources to the tasks for building a realistic project schedule. Cad Mate online video training course explains the features like evaluating the schedule, resource workloads, and managing budget along with bringing a project on time.

Topics include

  • Navigating the User Interface
  • Configuring preferences
  • Creating a project with Task
  • Maintaining calendars
  • Creating a work breakdown structure (WBS)
  • Creating and formatting a schedule
  • Formatting Gantt charts
  • Adding constraints
  • Setting a baseline
  • Assigning resources
  • Handling Cost
  • Printing reports
  • Importing and exporting projects

Duration: 15 Hrs.
